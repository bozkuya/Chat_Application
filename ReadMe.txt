To install the app
---------------------------
Create the project and install the required dependencies using Composer:
composer require slim/slim "^4.0"
composer require slim/psr7
composer require php-di/php-di
run database.sql and main.php files
---------------------
To test
-------------------
First, install PHPUnit via composer:
composer require --dev phpunit/phpunit
Run ChatAppTest.php
--------------------------
Thank you :)
